Resonance Music

Purpose: Assignment for my web course

The Process

This website was part of my coursework at Seneca College. I decided to combine my interest in music with the possiblity of creating a make-believe brand that sells quality CD's and vinyls.

How it began...

The main goal of this assignment was to use JavaScript to dynamically replicate the information cards for each album. This project uses three main functions

  • an onLoad function
  • a createCard function
  • a display function

These functions do all the heavy lifting by dynamically creating the cards rather than having to hard-code ALL the albums and their cards. This also means albums can be added to the source without needing to edit the source code.

What I learnt...

Starting off, this project helped me further understand JSON objects, their structures and how to manipulate them as needed. This project also emphasised how important it is to develop efficent code (even though this website may not be the most efficient). Furthermore, it taught me how to dynamically replicate elements (obviously...)

Technologies Used

  • HTML5
  • JavaScript
  • JSON manipulation

Personal Portfolio

Purpose: Personal portfolio website

The Process

This website is my personal portfolio. I use this space to display my qualifications, projects and more to anyone interested!

How it began...

So I decided that the one thing any aspiring programmer needs is a portfolio site. I began by creating some mockups on Figma. If you look at my first iteration of this website, you'd be surprised at how bad at web design I was. After creating what looked like a 6th grade school kid's poster, I decided to scrap that Figma project and start anew.

What I learnt...

The first question I asked was how do I make this seem personal. My website lacked character. After much introspective thought, I couldn't reach a conclusion. So, rather than being convoluted, I decided to look on the surface for inspiration and there it was... I'm Pakistani. That's where the idea of the سلام came from. I speak Urdu and it made sense to add some unique-ness through the language I spoke. From there, I built on the design as best as I could

Technologies Used

  • Bootstrap V5.3
  • jQuery
  • JavaScript
